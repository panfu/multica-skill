# Multica CLI Skill

A skill file for AI coding agents (Claude Code, OpenClaw, Hermes, etc.) to operate [Multica](https://multica.ai) via CLI.

## Install

Copy the prompt below and paste it into any AI coding agent session:

> Fetch https://raw.githubusercontent.com/panfu/multica-skill/main/SKILL.md and create a local skill called `multica` from it.

That's it. The agent will read the file and set up the skill accordingly.

## What It Covers

- Agent CRUD & skill assignment
- Issue lifecycle (create, update, assign, status, search)
- Issue comments & execution history
- Project & workspace management
- Runtime monitoring & token usage
- Daemon start/stop/logs
- Skill import & file management
